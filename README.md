## ⚙️ Getting Started

### Prerequisites

- Modern web browser (e.g., Chrome, Firefox, Edge).

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/INYAT-gif/Inyat.git
   cd Inyat
   ```

2. Open the `index.html` file in your browser to view the portfolio.
